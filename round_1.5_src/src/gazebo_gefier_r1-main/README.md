## Gazebo Simulation Overview (GEFIER_R1)

This README provides a high-level overview of the Gazebo Sim setup for the `GEFIER_R1` robot and points to the detailed guides in this folder.

---

## 1. What You Can Do with Gazebo

With this Gazebo Sim setup you can:

- **Install and verify Gazebo Sim** on your system.
- **Visualize the MINI_R1_V1 URDF** in Gazebo and RViz2.
- **Teleoperate the robot** (keyboard or joystick) and inspect **LiDAR data**.
- **View camera and depth feeds** via `rqt` and `rqt_image_view`.
- **Test the robot in multiple worlds** (warehouse, empty world, and custom SDF worlds).
